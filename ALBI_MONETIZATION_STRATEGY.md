## 🚀 최적 전략: 3단계 접근

### Phase 1: 웹/PWA로 시작 (즉시~3개월) ⚡

**목표**: 시장 검증 + 초기 수익 확보

```
1. PWA 구현 (1-2시간)
   ↓
2. 마케팅 시작
   - Google 검색 광고 (₩50만/월)
   - 대학 커뮤니티 마케팅 (무료)
   - SNS 광고 (₩30만/월)
   ↓
3. 목표: 월 100-200명 유료 사용자
   ↓
4. 순수익: ₩500,000 ~ ₩1,280,000/월
```

**핵심**: 
- 앱스토어 수수료 ₩0 → **100% 수익**
- 빠른 반복 개선 → 제품 완성도 ↑
- 사용자 피드백 즉시 반영

### Phase 2: 마케팅 채널 확대 (3-6개월) 📈

**월 ₩100만+ 수익 달성 시**:

1. **바이럴 마케팅**
   - 대학 동아리 제휴
   - 인플루언서 협업
   - 추천 프로그램 (친구 초대 시 할인)

2. **B2B 영업**
   - 대학교 취업지원센터
   - 취업 학원
   - 기업 인사팀 (신입 채용용)

3. **콘텐츠 마케팅**
   - 블로그 (면접 팁, 이력서 작성법)
   - 유튜브 (AI 면접 후기)
   - 성공 사례 공유

**목표**: 월 500-1,000명 유료 사용자

### Phase 3: 앱스토어 진출 고려 (6개월+) 🏪

**조건이 충족되면 진행**:
- [x] 월 순수익 ₩300만+ (안정적)
- [x] 사용자 1,000명+ (충성도 높음)
- [x] 제품 완성도 95%+
- [x] 예산 ₩300만 확보

**이때 Capacitor로 전환**:
```

↓
```
