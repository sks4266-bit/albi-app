## 🎯 주요 기능

### **1. 주휴수당 자동 계산**

#### **주 15시간 미만**
```
주휴수당: 0원
설명: "주 15시간 이상 근무하면 주휴수당을 받을 수 있습니다!"
```

#### **주 15-40시간**
```
주휴시간 = (주간 근무시간 ÷ 40) × 8시간
주휴수당 = 시급 × 주휴시간

예: 20시간 근무
주휴시간 = (20 ÷ 40) × 8 = 4시간
주휴수당 = 10,320원 × 4시간 = 41,280원
```

#### **주 40시간 이상**
```
주휴시간 = 8시간 (고정)
주휴수당 = 시급 × 8시간

예: 40시간 근무
주휴수당 = 10,320원 × 8시간 = 82,560원
```

### **2. 월급 예상 계산**
```
월급 = 주급 × 4.345주

이유: 
1년 = 365일
1주 = 7일
평균 주수 = 365 ÷ 7 ÷ 12 = 4.345주
```

### **3. 상세 설명 제공**
```
📊 계산 방식:

1. 주간 기본급
   10,320원 × 20시간 = 206,400원

2. 주휴수당 (비례 계산)
   주휴시간 = (20시간 ÷ 40) × 8시간 = 4시간
   10,320원 × 4시간 = 41,280원

3. 주급 합계
   206,400원 + 41,280원 = 247,680원

4. 예상 월급 (4.345주 기준)
   247,680원 × 4.345 = 1,076,170원
```
