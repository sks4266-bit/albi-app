# 알비(ALBI) 비즈니스 모델 및 성장 전략

## 🎯 비전
**"세계 최초 1시간 직장체험 플랫폼으로 글로벌 구인구직 시장을 혁신한다"**

## 💰 수익 모델 (Revenue Model)

### 1. 매칭 수수료 (핵심 수익원)

#### **가격 정책 (2026년 기준)**

| 구분 | 정규직 | 계약직 (3개월+) | 시간제 알바 | 1시간 체험 |
|-----|--------|----------------|------------|----------|
| **구인자 수수료** | 연봉의 10% | 계약금의 8% | 월급의 15% | 시급의 50% |
| **구직자 수수료** | 무료 | 무료 | 무료 | 무료 |
| **최소 금액** | 500,000원 | 200,000원 | 50,000원 | 10,000원 |
| **최대 금액** | 5,000,000원 | 1,000,000원 | 300,000원 | 50,000원 |

**예시**:
- 시급 10,000원 알바 매칭 → 구인자 50,000원 지불
- 월급 3,000,000원 알바 매칭 → 구인자 450,000원 지불 (15%)
- 연봉 40,000,000원 정규직 매칭 → 구인자 4,000,000원 지불 (10%)

#### **1시간 체험 특화 전략**
```
체험 요금 구조:
- 체험 시급: 구인자가 설정 (보통 10,000~30,000원)
- 알비 수수료: 체험 시급의 50%
- 구직자 수령: 체험 시급의 100% (알비가 먼저 지급, 구인자에게 청구)

예시:
체험 시급 20,000원 → 구인자 총 비용 30,000원 (시급 + 수수료 10,000원)
```

**KPI**:

- 글로벌 월 거래액: 500억원

### Phase 4: 시장 지배 (2031+)

## 🏗️ 기술 아키텍처 전략

### 1. 확장 가능한 인프라
```
현재 (MVP): Cloudflare Pages/Workers
↓
Phase 1 (10만 MAU): Cloudflare + Cloudflare D1
↓
Phase 2 (100만 MAU): 
- Frontend: Cloudflare CDN
- Backend: AWS ECS (Container)
- Database: AWS RDS (PostgreSQL Multi-AZ)
- Cache: AWS ElastiCache (Redis)
- Search: AWS OpenSearch
↓
Phase 3 (1000만 MAU):
- Multi-Region Deployment (서울, 도쿄, 싱가포르, 버지니아, 프랑크푸르트)
- Microservices Architecture
- Event-Driven (AWS EventBridge, SQS)
- Real-time Streaming (Kafka)
- AI/ML Pipeline (SageMaker)
```

### 2. 데이터베이스 전략
```
Phase 1: Cloudflare D1 (SQLite)
Phase 2: PostgreSQL (Single Region)
Phase 3: 
- Write: PostgreSQL Primary (Seoul)
- Read: PostgreSQL Read Replicas (Multi-Region)
- Cache: Redis Cluster
- Analytics: ClickHouse / Snowflake
Phase 4:
- Sharding by Region/Country
- CQRS Pattern
- Event Sourcing
```

### 3. AI/ML 전략
```
현재: Basic Keyword Matching
Phase 1: 
- GPT-4 기반 이력서 분석
- 성향 매칭 알고리즘
Phase 2:
- Custom LLM Fine-tuning
- 예측 매칭 (매칭 성공률 90%+)
Phase 3:
- Real-time Video Interview AI
- 자동 스킬 평가
- 급여 예측 모델
Phase 4:
- 완전 자동화 매칭
- AI HR Assistant
```

## 🎯 핵심 성공 요인 (Key Success Factors)

### 1. 네트워크 효과
- 구인자 많음 → 구직자 유입
- 구직자 많음 → 구인자 유입
- 선순환 구조

### 2. 데이터 축적
- 매칭 데이터 → AI 학습
- AI 정확도 향상 → 매칭 성공률 증가
- 경쟁사 진입장벽

### 3. 브랜드 신뢰도
- 안전한 거래 보장
- 분쟁 조정 시스템
- 보험 연계

### 4. 글로벌 확장 속도
- First Mover Advantage
- 지역별 맞춤 전략
- M&A 통한 빠른 확장
