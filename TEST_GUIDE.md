## 1️⃣ 아이디 찾기 테스트

### 접근 경로
```
로그인 페이지 → "아이디 찾기" 링크 클릭
또는 직접 접속: https://albi-app.pages.dev/find-id.html
```

### 테스트 시나리오 A: 정상 케이스
**입력**:
- 이름: `박지훈` (실제 가입한 이름)
- 휴대폰: `010-4459-4226` (실제 가입한 번호)

**예상 결과**:
```
✅ 아이디를 찾았습니다!
회원님의 아이디는 [사용자 아이디] 입니다.
```

**확인 사항**:
- ✅ 아이디가 정확히 표시되는가?
- ✅ "비밀번호 찾기" / "로그인하기" 버튼이 작동하는가?

### 테스트 시나리오 B: 오류 케이스
**입력**:
- 이름: `홍길동` (존재하지 않는 사용자)
- 휴대폰: `010-1234-5678`

**예상 결과**:
```
❌ 아이디를 찾을 수 없습니다
입력하신 정보와 일치하는 계정이 없습니다.
이름과 휴대폰 번호를 다시 확인해주세요.
```

## 2️⃣ 비밀번호 찾기 테스트

### 접근 경로
```
로그인 페이지 → "비밀번호 찾기" 링크 클릭
또는 직접 접속: https://albi-app.pages.dev/find-password.html
```

### 테스트 시나리오 A: 정상 케이스 (3단계)

#### Step 1: 이메일 + 아이디 입력
**입력**:
- 이메일: `skek4266@naver.com` (실제 가입한 이메일)
- 아이디: `skek4266` (실제 가입한 아이디)

**예상 동작**:
1. "인증 코드 발송" 버튼 클릭
2. 알림: "인증 코드가 이메일로 발송되었습니다!"
3. Step 2로 자동 이동

**F12 콘솔 확인**:
```javascript
📧 [이메일 발송]
수신: skek4266@naver.com
제목: [알비] 비밀번호 재설정 인증 코드
내용: 안녕하세요, 박지훈님.

비밀번호 재설정을 위한 인증 코드는 다음과 같습니다:

123456 (예시)

이 코드는 5분 동안 유효합니다.
```

#### Step 2: 인증 코드 입력
**입력**:
- 인증 코드: `123456` (콘솔에 표시된 6자리 코드)

**예상 동작**:
1. 인증 코드 입력
2. "인증 확인" 버튼 클릭
3. Step 3으로 자동 이동
4. 타이머 중지

**확인 사항**:
- ✅ 유효시간 타이머가 작동하는가? (5:00 → 4:59 → ...)
- ✅ "인증 코드 재발송" 버튼이 작동하는가?

#### Step 3: 새 비밀번호 설정
**입력**:
- 새 비밀번호: `newpassword123`
- 비밀번호 확인: `newpassword123`

**예상 동작**:
1. 비밀번호 입력 (8자 이상)
2. "비밀번호 변경" 버튼 클릭
3. 알림: "✅ 비밀번호가 성공적으로 변경되었습니다! 로그인 페이지로 이동합니다."
4. `/login.html`로 자동 리다이렉트

**최종 검증**:
- 로그인 페이지에서 새 비밀번호로 로그인 시도
- ✅ 로그인 성공하는가?

### 테스트 시나리오 B: 에러 케이스

#### B-1: 존재하지 않는 사용자
**입력**:
- 이메일: `notexist@test.com`
- 아이디: `notexist123`

**예상 결과**:
```
❌ 일치하는 계정을 찾을 수 없습니다
```

#### B-2: 잘못된 인증 코드
**입력**:
- 인증 코드: `000000` (잘못된 코드)

**예상 결과**:
```
❌ 인증 코드가 일치하지 않습니다
```

#### B-3: 비밀번호 불일치
**입력**:
- 새 비밀번호: `password123`
- 비밀번호 확인: `password456`

**예상 결과**:
```
❌ 비밀번호가 일치하지 않습니다
```

#### B-4: 비밀번호 길이 부족
**입력**:
- 새 비밀번호: `1234567` (7자)

**예상 결과**:
```
❌ 비밀번호는 최소 8자 이상이어야 합니다
```

#### B-5: 인증 코드 만료 (5분 초과)
**동작**:
1. 인증 코드 발송 받기
2. 5분 이상 대기
3. 만료된 코드 입력

**예상 결과**:
```
❌ 인증 시간이 만료되었습니다. 다시 시도해주세요.
(자동 새로고침)
```

   - 새 비밀번호 설정: `newtest456`
   - ✅ 비밀번호 변경 성공

4. **로그인** (`/login.html`)
   - 아이디: `testuser`
   - 비밀번호: `newtest456` (변경된 비밀번호)
