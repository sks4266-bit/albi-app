## 🐛 문제 상황

### 1. 하단 네비게이션과 푸터 겹침
- **증상**: 스토어 페이지 하단에 고정된 네비게이션 바가 푸터와 겹쳐 보기 어려움
- **원인**: `fixed bottom-0` 스타일로 하단에 고정된 네비게이션이 푸터 위에 덮임
- **영향**: 사용자가 푸터의 회사 정보 및 링크를 볼 수 없음

### 2. 포인트 적립 유도 부족
- **증상**: 포인트 잔액 카드에 간단한 메시지만 표시
- **원인**: 포인트 적립 방법에 대한 명확한 안내 부족
- **영향**: 사용자가 포인트를 어떻게 모으는지 알기 어려움

## ✅ 해결 방법

### 1. 하단 네비게이션 제거

**삭제된 코드**:
```html
<!-- 하단 네비게이션 - 삭제됨 -->
<div class="fixed bottom-0 left-0 right-0 bg-white border-t border-gray-200 p-4">
  <div class="max-w-md mx-auto flex justify-around">
    <a href="/">🏠 홈</a>
    <a href="/jobs.html">📍 내주변</a>
    <a href="/community.html">💬 커뮤니티</a>
    <a href="/store.html">🎁 스토어</a>
  </div>
</div>
```

**변경된 스타일**:
- `pb-20` → 제거 (하단 패딩 불필요)
- `fixed bottom-0` 네비게이션 → 완전 제거
- 푸터가 자연스럽게 페이지 하단에 표시됨

### 2. 포인트 잔액 카드 개선

**기존 디자인**:
```html
<div class="...">
  <p>보유 포인트</p>
  <h2>0 P</h2>
  <p>체험하고 일하면서 포인트를 모아보세요!</p>
</div>
```

**개선된 디자인**:
```html
<div class="...">
  <p>보유 포인트</p>
  <h2>0 P</h2>
  <div class="bg-white bg-opacity-20 rounded-lg p-3">
    <p>💰 포인트 적립 방법</p>
    <p>회원가입 50P • AI면접 20P • 체험완료 30P</p>
  </div>
</div>
```

**추가 요소**:
- 반투명 흰색 배경으로 가독성 향상
- 주요 적립 방법 3가지 간략 표시
- 이모지로 시각적 강조

### 3. 포인트 적립 안내 배너 추가

**새로운 배너**:
```html
<!-- 포인트 적립 안내 배너 -->
<div class="bg-gradient-to-r from-blue-500 to-purple-500 rounded-xl p-4 mb-6">
  <div class="flex items-center justify-between">
    <div>
      <p>🎯 포인트가 부족하신가요?</p>
      <p>1시간 체험으로 최대 50P 받으세요!</p>
    </div>
    <a href="/jobs" class="bg-white text-purple-600 px-4 py-2 rounded-lg">
      체험하기
    </a>
  </div>
</div>
```

**특징**:
- 눈에 띄는 그라데이션 배경 (파란색 → 보라색)
- 명확한 CTA (Call-to-Action) 버튼
- 포인트 부족 시 즉시 체험 페이지로 이동 가능

### 4. 포인트 부족 모달 개선

**기존 내용**:
```
💡 1시간 체험을 완료하면
최대 50P를 획득할 수 있어요!
```

**개선된 내용**:
```html
<div class="bg-blue-50 p-4 rounded-lg">
  <p>💡 포인트 적립 방법</p>
  <div>
    • 회원가입: 50P 즉시 지급
    • AI 면접 완료: 20P 적립
    • 체험 완료: 30P 적립
    • 구인자 평가: 최대 50P 보너스
    • 친구 초대: 친구 1명당 20P
  </div>
</div>
<p>지금 바로 체험하고 포인트를 모아보세요!</p>
```

**개선 사항**:
- 모든 포인트 적립 방법 상세 표시
- 각 활동별 포인트 금액 명시
- 행동 유도 메시지 추가
