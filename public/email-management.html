자동 실행 설정 (Cloudflare Cron Triggers)

매일 자동으로 구독 상태를 체크하고 알림을 발송하려면 Cloudflare Cron Trigger를 설정하세요.

wrangler.jsonc 설정:

{
  "triggers": {
    "crons": ["0 9 * * *"]  // 매일 오전 9시 (UTC)
  }
}

또는 수동으로 GET /api/check-subscriptions 호출
