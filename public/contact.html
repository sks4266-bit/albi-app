Q 🎤 AI 면접은 어떻게 진행되나요?

AI 면접은 실제 면접과 동일한 환경에서 연습할 수 있는 시스템입니다:

  1. 1️⃣ 직무 선택: 원하는 직무/업종 선택 (서비스직, IT, 영업 등)
  2. 2️⃣ 지역 설정: 면접 지역 선택
  3. 3️⃣ 면접 시작: AI가 실시간으로 질문 (10-15개)
  4. 4️⃣ 답변 입력: 텍스트 또는 음성으로 답변
  5. 5️⃣ 즉시 피드백: 각 답변에 대한 상세 분석
  6. 6️⃣ 종합 평가: 의사소통, 전문성, 태도 등 5개 영역 점수

Q 총 근무 시간은 어떻게 계산하나요?

공고 등록 시 입력하신 정보를 기반으로 자동 계산됩니다:

총 근무 시간 = (하루 근무 시간) × (근무 일수)

• 예시 1: 하루 4시간 × 5일 = 20시간 → 30,000원 (초단기 최대)

• 예시 2: 하루 8시간 × 20일 = 160시간 → 30,000원 (단기)

• 예시 3: 하루 8시간 × 30일 = 240시간 → 50,000원 (장기)

💡 공고 등록 페이지에서 자동 계산기를 제공하니 편리하게 이용하세요!

> Q ⭐ 신뢰도 점수는 무엇인가요?

신뢰도 점수는 구직자의 성실성과 신뢰성을 나타내는 지표입니다. (기본 100점)

📈 점수가 올라가는 경우:

  • 체험 성실히 완료
  • 정시 출근 기록
  • 고용주로부터 좋은 평가
  • 계약 이행 완료

📉 점수가 내려가는 경우:

  • 노쇼(무단 결근) - 관리자 승인 시 점수 차감
  • 반복적인 지각
  • 계약 중도 이탈
  • 고용주로부터 나쁜 평가

⚠️ 신뢰도 50점 미만 시:

체험 신청이 자동으로 제한됩니다. 고객센터에 문의하여 신뢰도를 회복하세요.

> Q 노쇼 신고는 어떻게 하나요?

구인자는 구직자가 약속 시간에 나타나지 않은 경우 노쇼 신고를 할 수 있습니다.

📝 신고 절차:

  1. 마이페이지 → 지원자 관리 탭 이동
  2. 해당 지원자의 "노쇼 신고" 버튼 클릭
  3. 노쇼 사유와 증거 입력 (메시지 스크린샷, 통화 기록 등)
  4. 신고 제출

⚙️ 처리 과정:

  1. 관리자가 신고 내용 검토 (1-2 영업일)
  2. 증거 확인 및 양측 입장 확인
  3. 승인 시: 구직자 신뢰도 점수 차감 (기본 -10점)
  4. 거절 시: 조치 없음

⚠️ 허위 신고 시 신고자에게 불이익이 돌아갈 수 있으니 신중하게 신고해주세요.

  • 부당한 노쇼 신고를 받은 경우
  • 특별한 사정이 있는 경우

Q 🎯 1시간 체험은 어떻게 신청하나요?

1시간 체험 신청 절차는 매우 간단합니다:

  1. 1️⃣ 공고 선택: 관심있는 일자리 공고 클릭
  2. 2️⃣ 체험 신청: "1시간 체험 신청" 버튼 클릭
  3. 3️⃣ 날짜 선택: 희망하는 체험 날짜와 시간 선택
  4. 4️⃣ 포인트 차감: 10 알비포인트 차감 (승인 거절 시 100% 환불)
  5. 5️⃣ 승인 대기: 구인자 승인 대기 (보통 24시간 이내)
  6. 6️⃣ 알림 수신: 승인 완료 시 알림톡/이메일 발송
  7. 7️⃣ 현장 방문: 약속된 시간에 현장 방문하여 관찰
ℹ️ 체험은 실제 일을 하는 것이 아니라 직장 분위기를 관찰하는 시간입니다!

  2. 2단계: 상단 메뉴에서 "공고 등록" 클릭
  3. 3단계: 필수 정보 입력
    • - 회사명, 업종, 근무지
    • - 모집 직종, 상세 업무 내용
    • - 시급, 근무 시간, 근무 기간
    • - 복리후생, 자격요건
  4. 4단계: 채용 유형 선택 (일반 체험 / ⚡바로채용)
  5. 5단계: 등록 완료 → 즉시 노출!

Q 🧾 세금계산서 발급이 가능한가요?

네! 세금계산서 자동 발급이 가능합니다:

📋 발급 방법
  1. 1. 마이페이지 > 결제 내역 접속
  2. 2. 해당 결제 건 선택
  3. 3. "세금계산서 발급" 버튼 클릭
  4. 4. 사업자 정보 입력 (자동 저장)
  5. 5. 발급 완료 → 이메일 자동 전송
⏰ 발급 시기
  • • 결제 후 7일 이내 신청 가능
  • • 매월 1~10일 사이 전월 발급 가능
  • • 승인 취소 시 자동 역발급
💼 사업자 회원은 결제 시 자동으로 세금계산서 발급 설정이 가능합니다!
