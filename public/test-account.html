고정 테스트 계정 (항상 로그인 가능)

아래 계정은 항상 로그인 가능한 고정 테스트 계정입니다.

  • 새로고침해도 계정이 유지됩니다
  • "새 계정 생성" 버튼으로 다른 테스트 계정으로 전환 가능
  • 총 3개의 고정 계정이 순환됩니다

아임포트 테스트 카드 정보

  • • 카드 번호: 5543-0400-0000-0009
  • • 유효기간: 임의 미래 날짜 (예: 12/25)
  • • CVC: 임의 3자리 (예: 123)
  • • 비밀번호: 앞 2자리 (예: 00)
  • • 생년월일: 6자리 (예: 900101)

테스트 사용 방법

로그인 테스트

  1. "새 계정 생성" 버튼 클릭
  2. "로그인 페이지로 이동" 버튼 클릭
  3. 생성된 테스트 이메일과 비밀번호 입력
  4. 로그인 후 서비스 이용

결제 테스트

  1. "새 계정 생성" 버튼 클릭
  2. "결제 페이지로 이동" 버튼 클릭
  3. 플랜 선택 (Basic/Standard/Premium/Unlimited)
  4. 생성된 테스트 이메일 입력
  5. "결제하기" → 아임포트 결제창에서 테스트 카드 입력
  6. "구독 관리 페이지"에서 구독 확인
