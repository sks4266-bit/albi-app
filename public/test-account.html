테스트 사용 방법

로그인 테스트

  1. "새 계정 생성" 버튼 클릭
  2. "로그인 페이지로 이동" 버튼 클릭
  3. 생성된 테스트 이메일과 비밀번호 입력
  4. 로그인 후 서비스 이용

결제 테스트

  1. "새 계정 생성" 버튼 클릭
  2. "결제 페이지로 이동" 버튼 클릭
  3. 플랜 선택 (Basic/Standard/Premium/Unlimited)
  4. 생성된 테스트 이메일 입력
  5. "결제하기" → 아임포트 결제창에서 테스트 카드 입력
  6. "구독 관리 페이지"에서 구독 확인
