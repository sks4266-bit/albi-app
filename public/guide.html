주요 기능 상세

1:1 AI 영상 면접

  • 개인화된 7개 질문 (사용자 프로필 기반)
  • 실시간 표정/시선/자세 분석
  • 제스처/음성/말하기 고급 분석 (Phase 5)
  • GPT-4 답변 감정 분석
  • 종합 평가 및 채용 가능성 분석
  • 면접 영상 자동 녹화 및 저장

패널 면접 (3명)

  • 3명의 전문 면접관 (HR/기술/경영진)
  • 면접관별 전문 분야 질문
  • 실시간 면접관 전환 애니메이션
  • 면접관별 개별 평가
  • 패널 종합 평가

내 면접 녹화

  • 모든 면접 자동 저장
  • 과거 면접 다시보기
  • 점수 비교 및 성장 추이
  • 영상 다운로드

고급 AI 분석 (Phase 5)

제스처 분석

MediaPipe Hands로 손동작 추적. 제스처 빈도, 움직임 강도, 자연스러움을 평가합니다.

음성 톤 분석

Web Audio API로 음높이, 음량, 안정성을 실시간 측정합니다.

말하기 분석

WPM(분당 단어 수), 휴지기, 필러 워드, 명확성, 유창성을 분석합니다.

감정 분석

GPT-4로 답변의 긍정성, 자신감, 논리성, 전문성을 평가합니다.

자주 묻는 질문

카메라/마이크가 작동하지 않아요

브라우저 설정에서 카메라와 마이크 권한을 허용해주세요. Chrome/Edge를 권장합니다.

면접 영상이 저장되나요?

네, 모든 면접은 자동으로 녹화되어 저장됩니다. '내 면접 녹화' 페이지에서 확인할 수 있습니다.

몇 번이나 면접을 볼 수 있나요?

무제한입니다. 원하는 만큼 연습하세요!

모바일에서도 사용할 수 있나요?

네, 모바일 브라우저에서도 사용 가능합니다. 단, 데스크톱 환경을 권장합니다.
